# ComfyUI Emissive Color Node

A custom ComfyUI node that extracts bright/emissive areas from an image, applies a blur, and outputs:
- A mask
- An emissive-colored image
- A solid-color version

## Installation

1. Navigate to your `ComfyUI/custom_nodes` folder.
2. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/comfyui-emissive-color.git
